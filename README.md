## Run Locally

Clone the project

```bash
  git clone https://github.com/codelikeagirl29/nextjs-portfolio
```

Go to the project directory

```bash
  cd nextjs-portfolio
```

Install dependencies with [npm](https://nodejs.org/en/download)

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

view at [localhost:3000](https://localhost:3000)
